WHAT IT DOES
------------
JK3DA Texel Kit is a Texel Density tool for Blender 4.0+.
It helps you measure, set and visually check Texel Density
across your 3D models – a standard step in game art and
film production pipelines.

Texel Density (TD) describes how many texture pixels land on
one centimetre of your model's surface. Consistent TD across
all objects in a scene ensures textures look uniformly sharp.

FEATURES
--------
- Scan Density     Measure TD for one or multiple objects at once
- Set Density      Scale UVs automatically to a target TD value
- Presets          One-click preset grid (20.48 / 10.24 / 5.12 / 2.56 / 1.28 / 0.64 px/cm)
- x0.5 / x2        Half or double current TD instantly
- Units            Switch between px/cm, px/m, px/in, px/ft
- Checker Preview  Assign a checker material to spot UV stretching
                   Supports custom checker maps (e.g. CustomUVChecker)
- Normalize UVs    Snap UVs that drifted outside 0-1 back into range
- Scale Warning    Alerts you if an object has unapplied scale (Ctrl+A)
- Scene TD Report  Scan all meshes in the scene at once, sorted by TD
                   Click any entry to select that object
- UV Editor Panel  Full panel available in both 3D Viewport and UV Editor

INSTALLATION
------------
1. Open Blender
2. Go to Edit > Preferences > Add-ons
3. Click "Install..." and select jk3da_texelkit.zip
4. Enable the add-on by checking the checkbox
5. Find the panel in the 3D Viewport under N-Panel > JK3DA Texel Kit
   and in the UV Editor under N-Panel > JK3DA Texel Kit

REQUIREMENTS
------------
- Blender 4.0 or higher
- Objects must have a UV map
- Apply object scale (Ctrl+A > Scale) before using Set Density

USAGE TIPS
----------
- Always apply scale (Ctrl+A) before scanning or setting TD
- Use the Scene TD Report to quickly find problem objects
- Load your own checker map via the image picker in Checker Preview
- In Edit Mode, enable "Selected Faces Only" to measure a subset
