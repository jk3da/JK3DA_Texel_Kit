# Changelog — JK3DA Texel Kit

All notable changes are documented here.  
Versioning follows [Semantic Versioning](https://semver.org/).

---

## [1.0.0] — 2026-05-10

First public release of the modular version.

### Added
- Texel density scan for single and multiple objects
- Selected-faces mode in Edit Mode
- Unit support: px/cm, px/m, px/in, px/ft
- Set density per island (EACH) or averaged (AVG)
- 6 preset densities (20.48 / 10.24 / 5.12 / 2.56 / 1.28 / 0.64 px/cm) plus Half / Double multipliers
- Normalize UVs operator (fits islands into 0–1 space)
- Checker preview: procedural JK3DA orange/dark pattern or custom texture
- Live checker scale slider
- Revert materials to original state
- Scene Report panel: scan all meshes in scene, sort by TD, click-to-select
- UV coverage % and world surface area (cm²) display
- Quality indicator: Too Low / Low / Good / High / Overkill
- Copy-to-target button for quick TD transfer between objects
- Panels in 3D View sidebar and UV/Image Editor
